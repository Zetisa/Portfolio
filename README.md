# Personal Portfolio

Hey! This is the source code for my personal portfolio website. I built this to document my journey as an Information Systems student at **UMN** and my time interning as a Data Scientist at **AirNav Indonesia**.

## 🛠️ The Stack
I kept it modern and minimal:
- **Framework:** Nuxt 3 (Vue.js)
- **Styling:** Tailwind CSS
- **Content:** Nuxt Content (for writing my research/blogs in Markdown)

## ✨ What's Inside?
I use this site to showcase:
- My research on **ADS-B Anomaly Detection** (handling that massive 235M record dataset).
- Technical insights I've picked up along the way.
- A clean, scroll-based UI that doesn't "jump" around.
