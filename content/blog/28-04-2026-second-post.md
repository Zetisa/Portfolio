# The AirNav Experience So Far

My internship at **AirNav Indonesia** has been more than just about analyzing aviation data; it’s been a crash course in modern technical workflows. Moving from an academic environment to a professional Data Science role meant I had to level up my "toolbelt" quickly.

### 1. Mastering the Basics: GitHub Workflow
Before this, GitHub was just a place to store code. Now, it’s my primary collaboration tool. I've learned how to manage repositories properly, handle version control for my research scripts, and ensure that my development process is structured and reproducible. It’s the backbone of everything I’m building here.

### 2. Research Efficiency with NotebookLM
One of the most interesting tools I’ve been introduced to is **NotebookLM**. In a field as complex as aviation security, I have to digest massive amounts of documentation and research papers. NotebookLM has been a game-changer for synthesizing this information, allowing me to "talk" to my data sources and get insights faster without losing the context.

### 3. Exploring the AI Frontier: Jules & Hermes (via Linux/WSL)
I’ve also been integrating specialized AI tools into my workflow:
* **Jules**: Instrumental in refining my thought process when navigating technical aviation protocols and complex documentation.
* **Hermes Agent**: This one is special. To run Hermes Agent effectively, I’ve been using **Linux (WSL)** on my Windows machine. 

Setting up a Linux environment specifically to handle Hermes Agent taught me a lot about environment management and why certain high-performance tools work better in a Unix-based system. It’s been a great experience bridging the gap between my primary OS and the specialized requirements of advanced AI agents.

### The Bigger Picture

Initially, I thought being a Data Scientist was just about writing Python code in a silo. But between these tools and the guidance from my supervisor, I’ve realized it's about building an efficient ecosystem. Whether it’s versioning code on GitHub or using AI agents within a Linux subsystem to speed up development, every tool serves a purpose in solving the bigger puzzle of aviation safety.

It's been a steep learning curve, but seeing how these tools translate into real-world impact at AirNav makes every bit of the effort worth it.
