# Hello, World!

This marks my very first post on this portfolio. To be honest, I never thought I’d find myself diving this deep into the frontend world, especially since my main focus has always been on **Systems Information** and **Data Science**. 

### Why Web Dev?

This project started because of a direct instruction from my supervisor during my internship at AirNav. He pointed out that I needed a proper way to document my research—like the ADS-B anomaly detection project I'm currently working on—instead of just letting it sit in local folders.

Initially, I saw it as just another task. But as I started working things out, I realized he was right: **data is only as good as how you communicate it.** A simple local file isn't enough to show the full picture of what I've been analyzing. That’s when the curiosity kicked in, and I decided to build this space from scratch using **Nuxt 3** and **Tailwind CSS**.

### The Learning Curve

I've got to admit, it was quite a ride. Even in these early stages, I found that exploring web development is surprisingly addictive. From wrestling with folder structures (which took me a while to get right, lol) to fine-tuning this monochrome, brutalist design—every bit of it feels like solving a complex puzzle.

There’s something satisfying about seeing your code turn into a visual interface. It’s a different kind of "logic" compared to training models or querying databases, but it’s definitely interesting.

### What's Next?

This website will be my digital playground. Expect more updates here—not just about web dev, but also about my internship journey, technical deep-dives into machine learning, and maybe some random thoughts on Information Systems.

It’s still a work in progress, but hey, we all have to start somewhere, right?

Stay tuned!
